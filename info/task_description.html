Enunciado

Beatriz, uma administradora de recursos humanos, recebe frequentemente formulários de inscrição onde os candidatos digitam seus nomes completos inteiramente em letras minúsculas. Para manter a consistência e a formalidade nos documentos oficiais da empresa, ela precisa de uma ferramenta que corrija automaticamente a capitalização dos nomes. Crie uma função que receba um nome completo com todas as letras em minúscula e retorne o nome com a primeira letra de cada parte do nome capitalizada, adequado para apresentação formal.

Entrada: Uma string (nome_completo) em minúsculas.

Saída: A string com cada parte do nome capitalizada.
